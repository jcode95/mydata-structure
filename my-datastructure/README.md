# datastructure
用Java实现的数据结构暨算法，同时对其进行了测试

准备工作
------
  1 为你的系统安装Maven  
  2 Setup环境 请在终端命令行上运行 “mvn clean install -DskipTests”， 然后再运行 “mvn eclipse:eclipse”,将项目Import到你的Eclipse中
